Objective : Don't Die


Tip : Use 80% Page zoom to see the whole screen, the screen is 780p, you can also tab off onto another tab to pause, dont leave this running in the background it runs off your computer not a website.


Controls : Using the normal WASD keys. R and . Reset so players who wanna idk reset faster can reset faster. The Arrow keys Jump in blocks of 50 pixels, so if you are in a tricky situation use this. Each wave adds 1 extra dot do goodluck.


How To Win : You Don't I'm working on a scoring system though for competition (:

BTW : Shifting is a form of moving which moves in jumps of 50 Pixels, Allowing you to Travel more space and pass through Blocks if Needed.

Right Layout Controls

Shift Up : Up Arrow

Shift Down : Down Arrow

Shift Right : Right Arrow

Shift Left : Left Arrow

. : Reset

Left Layout Controls

Move Foward : W

Move Backward : S

Turn Right : D

Turn Left : A

R : Reset
